# QA Checklist

Use this checklist before final delivery.

## Mobile
- [ ] Layout is usable on mobile widths (320px to 430px).
- [ ] No horizontal overflow appears.
- [ ] Buttons and links are easy to tap.
- [ ] Typography remains readable on small screens.

## Links
- [ ] Header and CTA links point to valid destinations.
- [ ] Footer links do not return 404 pages.
- [ ] External links open correctly.

## Spelling
- [ ] Brand name is consistent across the page.
- [ ] Headline/subheadline text has no spelling mistakes.
- [ ] Feature, testimonial, and FAQ text is proofread.

## Performance
- [ ] Build completes without errors.
- [ ] Images/assets are optimized for web delivery.
- [ ] No obvious layout shift during page load.

## Accessibility Basics
- [ ] Page has one main H1 heading.
- [ ] Buttons and links have clear labels.
- [ ] Color contrast is readable for body text and CTAs.
- [ ] Interactive elements are keyboard reachable.
